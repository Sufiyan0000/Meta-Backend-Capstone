The Meta Backend Developer Capstone is the final project in the Meta Backend Developer Professional Certificate program on Coursera. It showcases a full backend application built using Python, Django, and REST API principles, applying all the key skills learned throughout the course.

In this capstone, I developed a fully functional restaurant reservation and menu management system (based on the Little Lemon project case study). The project demonstrates my ability to design and implement secure, scalable, and data-driven backend solutions.
